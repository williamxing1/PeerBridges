# Project Overview

A web application for managing a volunteer-based one-on-one English tutoring program.

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Supabase PostgreSQL
- Supabase Auth
- Supabase Storage

---

## Requirements Documents

- English Requirements
- Chinese Requirements

When there is a conflict, ask for clarification instead of making assumptions.

---

# Core Features

## Authentication

Users can register and log in.

Roles:
- Student
- Tutor
- Administrator

---

## Student Features

### Scheduling
- View tutor availability
- Schedule lessons
- Cancel lessons
- View upcoming lessons

### Lesson History
- View completed lessons
- View teacher evaluations
- View homework and lesson notes

### Resources
- Access learning materials

---

## Tutor Features

### Availability
- Create available time slots
- Edit available time slots
- Remove available time slots

### Lesson Management
- View scheduled lessons
- Write class evaluations
- Record lesson notes and homework

---

## Administrator Features

### Reporting
- Student attendance reports
- Tutor attendance reports
- Student growth statistics
- Tutor growth statistics

### Management
- View all users
- View all lessons
- View all reports

### Communication
- Send messages to students and tutors

---

# MVP

Focus on these features first:

1. Authentication
2. Student lesson scheduling
3. Tutor availability management
4. Tutor class evaluations
5. Administrator reporting dashboard

Everything else is secondary.
